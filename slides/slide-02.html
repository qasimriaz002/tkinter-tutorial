Getting Started with Tkinter

To create a GUI, we need a "Root" window. This acts as the container for all other widgets (buttons, labels, etc.).

Key Concept: The mainloop() method is crucial. It keeps the window open and listens for events (clicks, key presses) indefinitely.

Without mainloop(), the script would execute, create the window, and close immediately.

(Live Window would appear here)

Empty Root Window
import tkinter as tk root = tk.Tk() root.title("My First App") root.geometry("400x300") root.mainloop()
