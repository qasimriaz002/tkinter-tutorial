Widget: Entry

The Entry widget allows the user to input a single line of text.

Key Methods:

  • .get(): Retrieves the current text.
  • .delete(0, END): Clears all text.
  • .insert(0, "text"): Inserts text at a specific index.
  • show="*": Used for password fields.

import tkinter as tk def submit(): name = entry.get() if name: result.config(text=f"Hello, {name}!") entry.delete(0, tk.END) root = tk.Tk() entry = tk.Entry(root, width=30, font=("Arial", 12)) entry.pack(pady=10) btn = tk.Button(root, text="Submit", command=submit) btn.pack() result = tk.Label(root, text="") result.pack() root.mainloop()
