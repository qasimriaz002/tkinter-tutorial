Widget: Text

Unlike Entry, the Text widget supports multi-line text input.

Usage Details:

  • Indexes use strings like "1.0" (Line 1, Char 0) or tk.END.
  • state="disabled" makes it read-only.
  • Often paired with a Scrollbar widget.

import tkinter as tk def get_content(): content = text_area.get("1.0", tk.END) print(content) root = tk.Tk() text_area = tk.Text(root, height=8, width=40) text_area.pack() text_area.insert(tk.END, "Type here... Line 2") btn = tk.Button(root, text="Show Content", command=get_content) btn.pack() root.mainloop()
