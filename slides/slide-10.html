Widget: Checkbutton

Checkbuttons are used for binary choices (On/Off). Tkinter requires a Variable object to track the state.

Variables: Use tk.IntVar() (stores 0 or 1) or tk.BooleanVar() to link the checkbox state to your Python logic.

Status: Unchecked

import tkinter as tk def show_status(): status = "Checked" if var.get() == 1 else "Unchecked" lbl.config(text=status) root = tk.Tk() var = tk.IntVar() cb = tk.Checkbutton(root, text="Subscribe", variable=var, onvalue=1, offvalue=0, command=show_status) cb.pack() lbl = tk.Label(root, text="Unchecked") lbl.pack() root.mainloop()
