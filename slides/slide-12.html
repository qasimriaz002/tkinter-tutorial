Widget: Scale

The Scale widget provides a slider to select a value from a numerical range.

Parameters:

  • from_: Minimum value.
  • to: Maximum value.
  • orient: "horizontal" or "vertical".
  • resolution: Step size.

Value: 50
import tkinter as tk def on_slide(val): lbl.config(text=f"Volume: {val}%") root = tk.Tk() lbl = tk.Label(root, text="Volume: 50%") lbl.pack() scale = tk.Scale(root, from_=0, to=100, orient='horizontal', command=on_slide, length=200) scale.pack() root.mainloop()
