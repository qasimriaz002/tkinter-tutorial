Widget: Listbox

A Listbox displays a list of items from which the user can select one or more.

Key Methods:

  • .insert(END, item): Add to list.
  • .curselection(): Returns tuple of selected indices.
  • .get(index): Get text at index.
  • selectmode: 'browse' (single) or 'extended' (multiple).

import tkinter as tk root = tk.Tk() lb = tk.Listbox(root, height=5, selectmode="single") lb.pack() items = ["Python", "Java", "C++", "JavaScript"] for item in items: lb.insert(tk.END, item) def show_selected(): selection = lb.curselection() if selection: print(lb.get(selection[0])) tk.Button(root, text="Show Selected", command=show_selected).pack() root.mainloop()
