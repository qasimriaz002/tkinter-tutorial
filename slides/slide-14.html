Widget: Spinbox

A Spinbox is an Entry widget with up/down arrows to cycle through a set of values (either a numerical range or a specific list of strings).

Usage:

  • Use from_ and to for numbers.
  • Use values=(...) tuple for text options.
import tkinter as tk root = tk.Tk() # Numeric Spinbox spin_num = tk.Spinbox(root, from_=0, to=10, increment=1, width=5) spin_num.pack(pady=5) # Text Spinbox spin_text = tk.Spinbox(root, values=("Small", "Medium", "Large"), width=10) spin_text.pack(pady=5) root.mainloop()
