Widget: Combobox (ttk)

The Combobox combines an Entry field with a drop-down list. It is part of the ttk (themed tk) submodule.

Key Features:

  • User can type a value OR select from the list.
  • .current(0) sets the default selection.

Selected: USA

from tkinter import ttk import tkinter as tk root = tk.Tk() options = ["USA", "UK", "India", "Germany"] combo = ttk.Combobox(root, values=options) combo.current(0) combo.pack(pady=10) def submit(): print(combo.get()) tk.Button(root, text="Submit", command=submit).pack() root.mainloop()
