Widget: Notebook (Tabs)

The Notebook widget manages a collection of windows/tabs. Excellent for organizing settings or complex forms.

Structure:

  • Create ttk.Notebook.
  • Create Frames for each tab content.
  • Add frames using .add(frame, text="Tab Name").

from tkinter import ttk import tkinter as tk root = tk.Tk() note = ttk.Notebook(root) tab1 = ttk.Frame(note) tab2 = ttk.Frame(note) ttk.Label(tab1, text="Settings Page").pack(padx=20, pady=20) ttk.Label(tab2, text="Profile Page").pack(padx=20, pady=20) note.add(tab1, text="Settings") note.add(tab2, text="Profile") note.pack(expand=True, fill="both") root.mainloop()
