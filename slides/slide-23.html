Displaying Images

Tkinter supports images (PNG, GIF) via PhotoImage (for basic formats) or PIL.ImageTk (for JPEG/PNG via Pillow).

CRITICAL BUG: If you assign PhotoImage to a local variable inside a function, it gets garbage collected before displaying. You must keep a reference to the image object.

(Placeholder Image Loaded)

import tkinter as tk root = tk.Tk() # 1. Load the image img = tk.PhotoImage(file="logo.png") # 2. Create Label with image img_label = tk.Label(root, image=img) # 3. Keep reference (prevents garbage collection!) img_label.image = img img_label.pack() root.mainloop()
