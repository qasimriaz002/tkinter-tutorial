Image Buttons

Buttons can display images instead of, or alongside, text. This is standard for toolbars.

The compound Parameter:

  • "left": Image left, text right.
  • "top": Image above text.
  • "center": Text over image.
import tkinter as tk root = tk.Tk() icon = tk.PhotoImage(file="save.png") btn = tk.Button(root, image=icon, text="Save File", compound="left", command=lambda: print("Saving...")) btn.image = icon # Prevent garbage collection btn.pack(pady=20) root.mainloop()
