Real-World Demo: Calculator

Building a calculator is a classic exercise to master grid() layout and event handling logic.

Logic: We append numbers to a string expression and use Python's eval() to compute the result.

Note on eval(): In production apps, eval() can be risky. For learning tools it's fine, but in real projects you should parse the string mathematically.

import tkinter as tk root = tk.Tk() root.title("Calculator") entry = tk.Entry(root, width=35, borderwidth=5, font=("Arial", 18)) entry.grid(row=0, column=0, columnspan=4, padx=10, pady=10) def button_click(n): entry.insert(tk.END, str(n)) def button_clear(): entry.delete(0, tk.END) def button_equal(): try: result = eval(entry.get()) entry.delete(0, tk.END); entry.insert(0, result) except: entry.delete(0, tk.END); entry.insert(0, "Error") buttons = ['7','8','9','/','4','5','6','*','1','2','3','-','C','0','=','+'] r, c = 1, 0 for t in buttons: cmd = lambda x=t: button_equal() if x=='=' else button_clear() if x=='C' else button_click(x) tk.Button(root, text=t, padx=40, pady=20, command=cmd).grid(row=r, column=c) c += 1 if c > 3: c, r = 0, r+1 root.mainloop()
