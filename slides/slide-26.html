Advanced Pack: Responsive Frames

This demonstrates how to build a classic layout: Header, Sidebar, and Main Content. Used in IDEs, email clients, and dashboards.

Key Concepts:

  • Header: Packed side="top", fill="x". Spans the width.
  • Sidebar: Packed side="left", fill="y". Spans the height.
  • Main Content: Packed side="left" with expand=True, fill="both". Takes remaining space.

Drag the resize handle on the demo to see how widgets adapt!

import tkinter as tk root = tk.Tk() root.geometry("400x300") top_bar = tk.Frame(root, bg="#ffcccc", height=30) top_bar.pack(side="top", fill="x") sidebar = tk.Frame(root, bg="#ccffcc", width=100) sidebar.pack(side="left", fill="y") main = tk.Frame(root, bg="#ccccff") main.pack(side="left", fill="both", expand=True) root.mainloop()
