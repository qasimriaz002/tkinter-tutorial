File Handling in Python

File handling allows you to create, read, update, and delete files stored on your disk. This is crucial for saving user data, application logs, or configuration settings.

1. Write Mode ("w")

Creates a new file or overwrites an existing one. Use this to initialize data.

2. Append Mode ("a")

Adds data to the end of the file without erasing existing content.

3. Read Mode ("r")

Opens a file for reading (default mode). Used to load saved data.

Note: Always use the with open(...) context manager. It automatically closes the file, preventing data corruption.

students.txt

John,22 Anna,24 William,31
# 1. WRITE MODE (Creates/Overwrites file) with open("students.txt", "w") as file: file.write("John,22\n") file.write("Anna,24\n") # 2. APPEND MODE (Adds to end of file) with open("students.txt", "a") as file: file.write("William,31\n") # 3. READ MODE (Reads line by line) with open("students.txt", "r") as file: for line in file: # Strip whitespace and split by comma data = line.strip().split(",") name = data[0] marks = data[1] print(f"Student: {name}, Marks: {marks}")
