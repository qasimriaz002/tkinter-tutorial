File Handling & Tkinter GUI

Connecting File I/O to a GUI allows users to persist data. When a user clicks a button, we trigger a file operation (Read/Write/Append) and update the widgets on screen.

Implementation Strategy

  • Separation of Logic: Keep file operations in specific functions (read_data, append_data).
  • Read: Open file in "r" mode, parse content, and display in a Text/Listbox widget.
  • Append: Get input from Entry widgets, open file in "a" mode, and write the new line.
  • Search: Read file line by line, match the name, and show specific result.
Tip: Always refresh the "View" list after adding a new student to show the updated file content.
import tkinter as tk from tkinter import messagebox # --- 1. FILE FUNCTIONS --- def append_student(name, marks): # Append mode adds to the end without overwriting with open("students.txt", "a") as file: file.write(f"{name},{marks}\n") def read_all_students(): content = "" try: with open("students.txt", "r") as file: content = file.read() except FileNotFoundError: content = "No records found." return content def search_student(name): with open("students.txt", "r") as file: for line in file: if name in line: return line.strip() return "Student not found." # --- 2. GUI FUNCTIONS --- def handle_add(): n = name_entry.get() m = marks_entry.get() if n and m: append_student(n, m) output_text.insert(tk.END, f"Added: {n}\n") def handle_view(): output_text.delete("1.0", tk.END) data = read_all_students() output_text.insert(tk.END, data) def handle_search(): query = name_entry.get() if query: output_text.delete("1.0", tk.END) result = search_student(query) output_text.insert(tk.END, result) # --- 3. TKINTER SETUP --- root = tk.Tk() root.title("Student Manager") name_entry = tk.Entry(root) marks_entry = tk.Entry(root) name_entry.pack(pady=5); marks_entry.pack(pady=5) tk.Button(root, text="Add", command=handle_add).pack(side=tk.LEFT, padx=5) tk.Button(root, text="View All", command=handle_view).pack(side=tk.LEFT, padx=5) tk.Button(root, text="Search", command=handle_search).pack(side=tk.LEFT, padx=5) output_text = tk.Text(root, height=10) output_text.pack(pady=10) root.mainloop()

Code Explanation:

1. File Functions

  • append_student: Opens students.txt in "a" (append) mode. This adds the new name and marks to the end of the file without deleting previous data.
  • read_all_students: Opens the file in "r" (read) mode. It uses a try...except block to handle the error if the file hasn't been created yet.
  • search_student: Loops through the file line by line. It checks if the searched name exists in the current line string and returns that line if found.

2. GUI Functions & Setup

  • name_entry.get(): Retrieves the text currently typed into the Name input box.
  • output_text.delete("1.0", tk.END): Clears the Text widget from the first character to the end before displaying new search results or the full list.
  • tk.Button: Three buttons are created, linking directly to handle_add, handle_view, and handle_search.
